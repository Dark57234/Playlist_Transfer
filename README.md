# Playlist Transfer

  Playlist transfer is an App that allows you to transfer playlists from one platform to another
  
</div>


> [!TIP]
> This application is still in progress and can only transfer playlists from Spotify to Youtube, and has a limit of 50 songs per playlist, if you try to download more tracks, the app will reach the limit of API requests.
> This application is for educational purposes only

### Pre-requisites
This application uses Spotify and Youtube Client ID's and Client Secrets to function. If you dont know how to get these, please click here for a tutorial on how to get them [How to get Client ID and Secret](https://github.com/Dark57234/Playlist_Transfer/tree/main/Setup) 

## Usage
![Playlist_Transfer usage image](https://github.com/Dark57234/Playlist_Transfer/blob/main/Welcome.png) 

When you open the application you will be greated and asked for the service you want to transfer your playlists to, afterwards add your Client ID and your Client Secret for both the platforms you're trying to transfer to. After these steps add your Spotify playlist link and after a bit your playlist will be succesfully added to your platform of choice. Enjoy :D
